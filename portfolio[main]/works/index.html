サイト名：PET HOUSE

https://mocopen3.com/04_23_Hashimoto(1)/index.html

担当：Design/Coding
ツール：Visual Studio Code/ MySQL/
開発言語：HTML5/ CSS/ PHP/ Javascript
フレームワーク：Bootstrap4

コメント：
内容：自分が飼っているペットの血統書番号を登録しておくことで、ペットの兄妹や親を探せたり、もしもペットが行方不明になってしまった場合に、このサイトで探すことが出来るというアプリを作成しました。
血統書を元に登録したデータはMySQLで管理し、PHPを使って登録や引き出しを行なっています。TOPページはBootstrapを利用し作成しました。
きっかけ：よく見ているYouTubeの動画内で、YouTubeを通してペットの生き別れした犬の兄弟が再開するというのがあって、その時に血統書のID番号が証拠となったという話を見たのがきっかけです。自分は犬を飼っていた経験がありましたが、この血統書の番号で兄妹かどうかを確認する方法をそれまで知らず、せっかくID番号で管理しているのならばもっと周知して横のつながりやコミュニティーがそこから生まれたら面白いだろうなと思いこのアプリを作りました。
苦労した部分：PHPを使って写真やデータをMySQLに登録する部分と今回初めてチャレンジで導入した、Bootstrapです。とにかくイメージした通りに配置や修正が上手くいかず苦労しましたが、沢山のサイトやYouTubeの動画も活用しながら自力で解決しました。
